# SAC — Scrili AntiCheat

перед презентацией античита, я был бы непротив добровольного пожертвования в мой проект :)
https://boosty.to/scril1

Античит для Minecraft 1.20.1 с поддержкой Forge.

Создан для серверов на Forge

## Чеки

| Чек | Описание |
|-----|----------|
| SpeedHack | Превышение скорости передвижения |
| Fly | Полёт без разрешения |
| Jesus | Хождение по воде |
| Reach | Удар на слишком большой дистанции |
| KillAura | Атака сквозь стены или без взгляда на цель |
| FastAttack | Слишком частые удары |
| AutoClicker | Превышение кликов в секунду |
| Scaffold | Слишком быстрая установка блоков |
| ModBlacklist | Запрещённые моды на клиенте |
| AntiESP | Скрытие игроков за стенами |
| AimBot | Детект аим-бота через движение мыши |
| Freecam | Взаимодействие с блоками на большой дистанции |

## Установка

**Forge сервер:**
Положи `sac-forge.jar` в папку `mods/` сервера и клиента.
Клиентский мод обязателен — без него AimBot detection не работает.

## Команды

| Команда | Описание |
|---------|----------|
| `/sac status` | Список игроков с варнами |
| `/sac status <игрок>` | История нарушений игрока |
| `/sac reset <игрок>` | Сброс варнов |
| `/sac reload` | Перезагрузка конфига |

## Конфиг

Конфиг генерируется автоматически при первом запуске.
Forge: `config/sac-server.toml`

Каждый чек можно включить или выключить отдельно.
